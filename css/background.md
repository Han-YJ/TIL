# Background

## repeat & position & size
```
background-repeat: no-repeat;
background-position: center;
background-size: cover;
```
- background-repeat
  - no-repeat: 반복을 없애고 지정된 이미지 사이즈만 보여줌
- background-position
  - center: 중간이미지가 보임
- background-size
  - cover: 크기만큼 채워져서 나옴

```
background: center/cover no-repeat url('url~')
```
- 이렇게 간단하게 사용가능
